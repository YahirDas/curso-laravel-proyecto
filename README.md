# SISTEMA_YAHIR

## Descripción
Este proyecto es una aplicación web desarrollada en PHP con el framework Laravel. El objetivo es implementar el modelo MVC en un proyecto sencillo parte del curso de INFOTEC de PHP.

## Tecnologías
* PHP 8.x
* Laravel 10/11
* MySQL
* CSS

## Instalación
Descarga el proyecto completo, ejecuta el servidor en el puerto 8000 asi como XAMPP.
